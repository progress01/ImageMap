# HTML Image Map Generator 
製作具備「點擊互動」與「說明提示」的圖片地圖 (Image Map)。
它支援**矩形**與**多邊形**繪製，畫完區域後可立即輸入說明文字與連結。最強大的功能是支援**「單檔匯出」**，將圖片與互動效果打包成一個獨立的 HTML 檔案，無需架設伺服器即可直接傳送給他人或進行簡報。

## ✨ 主要功能

* **純前端運作**：所有處理皆在瀏覽器完成，圖片不需上傳伺服器，安全且快速。
* **多種形狀支援**：
    * 🟦 **矩形 (Rect)**：適合規則區域。
    * 🔷 **多邊形 (Poly)**：適合不規則形狀（如地圖分區）。
* **即時資料輸入**：繪製完畢後，自動彈出視窗輸入**「說明文字」**與**「連結網址」**。
* **視覺化回饋**：編輯時可直接在畫布上看到區域名稱，避免遺忘。
* **單檔匯出 (Standalone Mode)**：
    * 一鍵下載包含圖片數據 (Base64) 的 HTML 檔。
    * 匯出的檔案具備 **SVG 互動層**：滑鼠移入會有高亮效果與浮動提示 (Tooltip)。
    * 支援點擊跳轉連結。

## 🚀 如何使用

### 1. 線上使用 (GitHub Pages)
[點擊這裡開啟工具]([https://progress01.github.io/ImageMap/])

### 2. 本地使用
1. 下載本專案的 `index.html`。
2. 直接雙擊檔案，使用瀏覽器 (Chrome, Edge, Safari) 開啟即可。

## 📝 操作流程

1.  **上傳圖片**：點擊「選擇檔案」載入圖片。
2.  **選擇工具**：點選「畫矩形」或「畫多邊形」。
3.  **繪製區域**：
    * **矩形**：拖曳滑鼠畫出框框，放開後**輸入說明與連結**。
    * **多邊形**：點擊多個頂點，**雙擊 (Double Click)** 結束繪製並輸入資料。
4.  **匯出成果**：
    * **簡報/分享用**：點擊 **「下載展示用 HTML」**，獲得一個可直接打開互動的檔案。
    * **開發者用**：展開下方的「查看生成的標準 HTML 代碼」複製 `<map>` 標籤。
